## 概要

シリアル接続でデータを受信しグラフを表示する。また同時にデータをファイルに記録することも可能。


![PIC Meter実行画面](readme_pics/pic-meter-main.jpg)
<br/>PIC Meter実行画面

![PIC Meter設定画面](readme_pics/pic-meter-dialog.jpg)
<br/>PIC Meter設定画面

**受信可能なシリアル出力例**

```
16.3
16.3
16.2
16.2
16.3
```
